Note that if the property is not defined, this method will do nothing. If the property is read-only, it will throw an exception.

{
    return method_exists($this, 'get' . $name) || $checkVars && property_exists($this, $name);
}

Returns a value indicating whether a property can be read. A property is readable if:

- the class has a getter method associated with the specified name (in this case, property name is case-insensitive); - the class has a member variable with the specified name (when `$checkVars` is true);
