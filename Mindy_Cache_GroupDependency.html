GroupDependency marks a cached data item with a group name.

You may invalidate the cached data items with the same group name all at once by calling [[invalidate()]].

protected function generateDependencyData($cache)
{
    if ($this->group === null) {
        throw new InvalidArgumentException('GroupDependency::group must be set');
    }
    $version = $cache->get([__CLASS__, $this->group]);
    if ($version === false) {
        $version = $this->invalidate($cache, $this->group);
    }
